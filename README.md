# CodeLeap Network

## O que foi usado

- **React** – interface
- **TypeScript** – tipagem
- **Vite** – build e dev server
- **ESLint** – lint do código
- **pnpm** – gerenciador de pacotes

## Funcionalidades

- Login com nome de usuário (salvo no navegador)
- Criar novo post (título e conteúdo)
- Listar posts em ordem cronológica
- Editar e excluir apenas os posts do usuário logado
- Paginação da lista (10 posts por página)

## Como rodar

```bash
# Instalar dependências
pnpm install

# Subir em desenvolvimento
pnpm dev
```

Build para produção:

```bash
pnpm run build
```
