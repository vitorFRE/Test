# CodeLeap Network

## O que foi usado

- **React** – interface
- **TypeScript** – tipagem
- **Vite** – build e dev server
- **ESLint** – lint do código
- **pnpm** – gerenciador de pacotes

## Como rodar

```bash
# Instalar dependências
pnpm install

# Subir em desenvolvimento
pnpm dev
```

Build para produção:

```bash
pnpm run build
```
